Kosho Sakae Auto Post

Automated video content posting web application

About This Service

Kosho Sakae Auto Post is a web-based application that automates video content posting through external video publishing APIs and OAuth authentication. It integrates with Make.com and Creatomate to automate the full workflow from content creation to publishing.

Key Features

  • Automated video publishing via OAuth-authorized accounts
  • AI-generated video content scheduling and posting
  • Integration with Make.com automation workflows
  • Integration with Creatomate for dynamic video creation
  • Secure OAuth 2.0 authentication flow

API Products & Scopes Used

This application uses the following API products and permission scopes:

Login Kit Content Posting API user.info.basic video.publish video.upload

Login Kit — OAuth 2.0 authentication for securely linking user accounts.
Content Posting API — Automated posting of video content to authorized accounts.

How It Works

Users grant permission through a standard OAuth 2.0 flow. Once authorized, the service operates strictly within the granted permission scope. Users can revoke access at any time through the platform's settings page.

All operations are performed on behalf of the user only with their explicit consent. No content is posted without prior authorization.
